# vue-chat
vue + socket.io + mongodb 实现的一个仿网页版的微信

# 拥有功能
 - 用户注册
 - 用户登录
 - 用户退出
 - 获取用户
 - 创建群组
 - 获取全部好友
 - 获取全部群组
 - 添加好友
 - 添加聊天群组
 - 搜索用户
 - 搜索群组
 - 获取群组成员
 - 删除好友
 - 删除群组
 - 推送消息
 - 拉取消息

# 目录内容
```
|-vue-chat-client   vue搭建的前端页面,监听8080端口
|-vue-chat-doument  开发文档
ㄴvue-chat-server   socket.io+monogdb搭建的服务端,监听3002端口
```
